CSS - Tutorial

Cascading Stylee Sheet

På den här sidan kommer vi gå igenom lite grunder i CSS och vad man kan göra. Det är ett språk som ändrar utseende på en webbsida skriven i ett markup språk t.ex HTML. Nedan kan du lära dig hur man ändrar färg, byter placering på något, ändrar form på ett objekt etc. Man kan skriva CSS på tre olika sätt inline, external, internal.


Vad din html-fil bör innehålla:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta http-equiv="X-UA-Compatible" content="ie=edge"> <title>Document</title> <link rel="stylesheet" href="style.css"> Detta är external stylesheet </head> <body> <style > Detta är internal CSS</style> <h1 style="color:blue;" > Detta är inline CSS</h1> </body> </html>

Det är rekomenderat att använda external stylesheet.

Ändra bakgrundsfärg

Med CSS kan man ändra färger på backgrunden. Testa att trycka på knapparna nedan för att ändra backgrundsfärgen på rutan nedan.

div{ background-color: white; }


Borders

Med egenskapen border kan du betona eller rama in text eller objekt. Nedan kan du se olika exempel på borders.

.border-red{ border: solid 2px red; } .border-dotted{ border: dotted 2px black; } .border-bottom{ border-bottom: solid 2px black; } .border-thick{ border: solid 10px black; }

Ändra Font

Med egenskapen font-family kan man ändra font i CSS. Egenskapen font-size ändrar storleken på texten. Man kan bland annat använda em eller px (pixlar). Om man skriver t.ex 2em betyder det att den kommer bli två gånger större än orginaltexten, medans 2px betyder att texten kommer bli två pixlar stor.

.font-times{ font-family: 'Times New Roman', Times, serif; font-size: 2.5em; } .font-courier{ font-family: 'Courier New', Courier, monospace; font-size: 2em; } .font-arial{ font-family: Arial, Helvetica, sans-serif; font-size: 1.5em; } .font-cambria{ font-family: Cambria, Cochin, Georgia, Times, serif; font-size: 1em; }

Ändra Placering || Float

Med egenskapen float kan man ändra position på innehållet på sidan. Nedan ser du hur den ena texten är vänsterplacerad och andra texten är högerplacerad.

.left { float: left; } .right { float: right; }

Ändra Form

Med egenskapen border-radius kan man ändra form på t.ex en bild

.img-form{ border-radius: 0%; }

Animering

I exemplet nedan har vi gett en div ett animation-attribut. Den tar en @keyframes: "mymove", en tidsperiod "5s" och hur många gånger animationen ska köras "infinite".

Med attributet @keyframes kan man animera i CSS. Det fungerar genom att man anger elementets start- och slutegenskaper: t.ex "starta som grön och sluta som blå".

.animation { width: 100px; height: 100px; border-radius: 100%; background: #0171bf; position: relative; animation: mymove 5s infinite; } @keyframes mymove { from { top: 0px; background: #0171bf; } to { top: 400px; background: #d44; } }
